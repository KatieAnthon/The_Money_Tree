## Getting Started

### Prerequisites:
1. Install NPM packages:
   `npm install`
   `npm install expo`
3. Enter your IP address in config.js
  `const IP = 'ENTER YOUR IP'`;

## Running the app
To see this running in expo from your phone:
`cd frontend`
`npm start`
